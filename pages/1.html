Replaced Elements in HTML

Element Example Code Description Preview
<img> <img src="https://via.placeholder.com/150" alt="Sample image"> Displays an image from an external source.
<video> <video src="video.mp4" controls></video> Embeds a video player (local video required).
<audio> <audio src="sound.mp3" controls></audio> Embeds an audio player (local audio required).
<iframe> <iframe src="https://example.com"></iframe> Embeds another webpage.
<embed> <embed src="file.pdf" width="200"> Embeds external resources like PDF or multimedia.
<object> <object data="file.swf" type="..."></object> Embeds multimedia or documents with fallback.
<canvas> <canvas id="myCanvas"></canvas> Used to draw graphics with JavaScript.
<input type="image"> <input type="image" src="btn.png"> Displays an image as a submit button.
